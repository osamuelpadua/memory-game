# Projeto Jogo da Memória



## O que é

O jogo da memória ‚ um clássico jogo formado por pe‡as que apresentam uma figura em um dos lados. 
Cada figura se repete em duas pe‡as diferentes. Para come‡ar o jogo, as peças são postas com as figuras voltadas para baixo, para que não possam ser vistas.
 Cada participante deve, na sua vez, virar duas peças e deixar que todos as vejam. Caso as figuras sejam iguais, o participante deve recolher consigo esse par e jogar novamente. 
 Se forem peças diferentes, estas devem ser viradas novamente, e sendo passada a vez ao participante seguinte. Ganha o jogo quem tiver mais pares no final do jogo.
 (https://pt.wikipedia.org/wiki/Jogo_de_mem%C3%B3ria)

 Este jogo é uma representação digital do clássigo jogo da memória descrito acima.
 

## Como jogar


* Escolha duas cartas clicando nelas para elas desvirarem;

* Se formarem um par, elas permanecerão desviradas. Caso contrário, elas serão viradas novamente;

* Encontre todos os pares no menor tempo possível

## Pontuação

* 1 estrela por encontrar todos os pares com mais de 30 jogadas;
* 2 estrelas por encontrar todos os pares em até 30 jogadas;
* 3 estrelas por encontrar todos os pares em até 16 jogadas;


## Estrutura do jogo

* HTML e CSS para o tabuleiro com as cartas
* Foi adicionado um temporizador que começa a partir da primeira jogada e para quando todos os pares são encontrados
* Foi adicionado um sistema de pontuação com estrelas
* Foi adicionado uma janela que parabeniza o jogador no final do jogo, apresenta as informações de pontuação e pergunta se o usuário deseja jogar novamente.
